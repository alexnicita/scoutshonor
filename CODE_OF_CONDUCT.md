## Contributor Covenant Code of Conduct

This project follows the Contributor Covenant. By participating, you agree to uphold this code.

- Be respectful and inclusive.
- Accept constructive feedback.
- Focus on what is best for the community.
- Show empathy and kindness to others.

Enforcement guidelines and the full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Reporting: Please report unacceptable behavior privately via GitHub Security Advisories or email the maintainer at alex@nicita.cc.
